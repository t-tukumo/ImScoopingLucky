# About
「[小説家になろう](https://syosetu.com/)」からランダムに表示するアプリです。

ページをめくるといきなり何かの作品の本文のどこかが表示されます。
ページの先頭にタイトルは表示されません。長いタイトルで消耗せずにスコップしたい人用。

お好みの検索フィルターを設定することができます。
